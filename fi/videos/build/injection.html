About injection molding

Plastic flakes are heated and injected into a mold. It’s a relatively quick process which is well suited for creating small objects repeatedly. You can make the molds yourself using CNC mills or lathes, or by simply welding them.

Technical drawings

You can find all the technical drawings to build this machine in our download-kit. They are divided into different steps and contain all the dimensions you need to build the machine.

Creations

This injection molding machine is well suited for making small detailed objects. Put some effort into your mold once and it will pay off in the end. Since it works with molds you can easily replicate it and set up a production.
