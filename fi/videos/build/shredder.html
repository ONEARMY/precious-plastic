About the shredder

Plastic waste is shredded into flakes which will be used in the other machines to create new things. You can select the output size of these flakes by changing the sieve inside the machine to create different patterns and processes.

Technical drawings

You can find all the technical drawings to build this machine in our download-kit. They are divided into different steps and contain all the dimensions you need to build the machine.

3D overview

Below you can find a 3D visual of the machine, just to give you a basic understanding how it works, click the button to turn it around. Detailed CAD drawings can be found in the download-kit.

Output

The output of the shredder isn’t super sexy just some chopped up plastic flakes. You can change the size of these flakes by changing the sieve. Different sizes of flakes will give a different pattern, smaller flakes melt easier but take longer to shred. Play around and see what works the best for you!
